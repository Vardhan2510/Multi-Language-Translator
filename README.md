# Multi-Language-Translator

A modern, responsive web application that provides instant translation across multiple languages using a clean and intuitive user interface.

## Features

- **Multi-Language Support**: Translate text between numerous global languages
- **Real-time Translation**: Instant translation as you type
- **User-Friendly Interface**: Clean, modern design with gradient background
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Copy to Clipboard**: Easily copy translated text with one click
- **Language Selection**: Simple dropdown menus for source and target language selection

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Modern CSS with gradient backgrounds and smooth transitions
- **API**: Integration with translation API for accurate translations

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for API calls

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Vardhan2510/Multi-Language-Translator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Multi-Language-Translator
   ```

3. Open `index.html` in your web browser:
   ```bash
   open index.html
   ```

## Usage

1. Select the source language from the first dropdown
2. Select the target language from the second dropdown
3. Enter the text you want to translate in the input field
4. The translation will appear in real-time in the output field
5. Click the copy button to copy the translated text to your clipboard
